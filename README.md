# Zen1 төсөл

## Төслийн тойм
Zen1 төсөл нь цахим худалдааны платформ бөгөөд хэрэглэгчид бүтээгдэхүүнүүдийг хайж, худалдаж авах, худалдахаас авах боломжийг олгодог. Энэхүү төсөл нь хэрэглэгчдийн хэрэгцээнд нийцсэн, хялбар ашиглах боломжтой системийг бий болгох зорилготой.

## Суулгалтын заавар
1. **Бусад шаардлагатай номын сангуудыг суулгана.** 
    - `npm install` гэж команд бичнэ.

2. **Серверийг эхлүүлнэ.** 
    - `npm start` гэж команд бичнэ.

3. **Браузер дээр `http://localhost:3000` -д нэвтрэнэ.** 

## Онцлог
- Хэрэглэгчийн бүртгэл болон нэвтрэлт
- Бүтээгдэхүүн хайх функц
- Бүтээгдэхүүн зарж борлуулах боломж
- Найдвартай төлбөрийн систем

## Архитектур
Zen1 систем нь **клиент-сервер** архитектур ашигладаг. Клиент талын код (React, Vue, эсвэл өөр) нь хэрэглэгчийн интерфэйсийг бүрдүүлж, сервер талын код (Node.js, Express) нь өгөгдлийн боловсруулалт, хадгалалтыг хариуцдаг.

## Хэрэглэгчийн үүрэг
- **Хэрэглэгч:** Нэвтэрч, бүтээгдэхүүн хайж, худалдан авалт хийх.
- **Зарлагч:** Бүтээгдэхүүн байршуулж, худалдагчийн үүрэг гүйцэтгэнэ.
- **Админ:** Системийн менежментийг хариуцна, бүтээгдэхүүнүүдийг хянах.
